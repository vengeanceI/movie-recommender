# üé¨ Movie Recommendation System

A content-based movie recommendation system built with Streamlit and deployed on Streamlit Cloud.

## Features
- Content-based filtering using TF-IDF vectorization
- Interactive web interface
- Real-time recommendations
- Genre-aware similarity scoring
- Responsive design

## Quick Start
1. Select a movie from the dropdown
2. Click "Get Recommendations"  
3. Explore similar movies!

## Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- **Frontend**: Streamlit
- **ML**: scikit-learn, pandas, numpy
- **Deployment**: Streamlit Cloud

## Dataset
Currently using sample data. Ready to integrate TMDB 5000 dataset.
